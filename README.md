# 🌾 CropCast AI

> ML-powered farming decisions for any location on Earth — backed by 33 years of NASA satellite data.

CropCast AI fetches real historical weather from the NASA POWER API, trains location-aware weather regressors, and runs a Random Forest crop classifier to recommend what to grow - for any lat/lon on the planet. Users pick a location on an interactive map, select a month (5 past → now → 5 future), and instantly see predicted weather, ML crop recommendations with confidence scores, and cultivation materials.

## ✨ Features

- 🗺️ **Global location picker** — click a map, search a city, or use GPS
- 🛰️ **NASA POWER integration** — 33 years of real satellite weather data per location
- 🤖 **ML weather regressors** — Gradient Boosting trained on real data (R² > 0.95)
- 🌱 **ML crop classifier** — Random Forest (99.5% accuracy) with confidence scores
- 📅 **Forecast window** — 5 months back · current · 5 months ahead (max 1 year future)
- 🌡️ **Weather cards** — temperature, rainfall, humidity, wind speed
- ✅ **Crop recommendations** — what to grow with % confidence bars
- ⚠️ **Crops to avoid** — season-aware avoid list
- 🧰 **Cultivation materials** — tools and inputs per season
- 📈 **Annual trend chart** — interactive yearly overview (rainfall / temp / humidity)
- 🏷️ **Data source badge** — shows NASA REAL vs SYNTHETIC clearly in the UI

## ⚙️ Setup & Run

### 1. Clone the repo
```bash
git clone https://github.com/Jayavarshini-Jayakumaran/CropCast-AI.git
cd CropCast-AI
```

### 2. Create virtual environment
```bash
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Download the crop dataset
Get **Crop Recommendation Dataset** from Kaggle:
https://www.kaggle.com/datasets/atharvaingle/crop-recommendation-dataset

Save as: `data/crop_recommendation.csv`

> This is the only manual download step. The file is excluded from git to keep the repo lightweight.

### 5. Fetch NASA weather data
```bash
# Default: Coimbatore, Tamil Nadu, India
python fetch_nasa_data.py

# Any location: python fetch_nasa_data.py <lat> <lon>
python fetch_nasa_data.py 13.08 80.27    # Chennai
python fetch_nasa_data.py 28.61 77.20    # Delhi
python fetch_nasa_data.py 1.35  103.82   # Singapore
```

### 6. Train the models
```bash
# Uses NASA CSV automatically if available, else falls back to synthetic data
python train_model.py

# Force synthetic data (offline / no internet)
python train_model.py --synthetic
```

### 7. Run the app
```bash
python app.py
```

Open **http://localhost:5000**

## 🌍 Changing Location (in the UI)

1. Click **Change** in the header location pill
2. Search a city name, click the map, or pick a preset
3. Click **Fetch NASA Data & Retrain Model**
4. Wait ~10 seconds — the model retrains live on the new location's data
5. All predictions update automatically — no restart needed

## 🤖 ML Models

| Model | Algorithm | Dataset | Accuracy |
|---|---|---|---|
| Crop Classifier | Random Forest (200 trees) | Kaggle crop_recommendation.csv — 2200 rows, 22 crops | 99.5% |
| Weather Regressors | Gradient Boosting (300 trees) | NASA POWER API — 33 years × 12 months per location | R² up to 0.90 |

**Crop classifier features:** N, P, K, temperature, humidity, pH, rainfall

**Weather regressor features:** month_sin, month_cos, year (cyclical encoding)

## 🛰️ Data Sources

| Source | What it provides | Cost |
|---|---|---|
| [NASA POWER API](https://power.larc.nasa.gov/) | 33yr monthly temperature, rainfall, humidity, wind — any lat/lon on Earth | Free, no login, no key |
| [Kaggle Crop Dataset](https://www.kaggle.com/datasets/atharvaingle/crop-recommendation-dataset) | Soil + weather → crop label, 2200 samples, 22 crops | Free (Kaggle account needed) |

> No API keys anywhere in this project. Safe to push to public GitHub as-is.

## 🔌 API Endpoints

| Endpoint | Method | Description |
|---|---|---|
| `/api/predict/<month>?year=<year>` | GET | ML prediction for a specific month/year |
| `/api/yearly?year=<year>` | GET | All 12 months weather data |
| `/api/location/meta` | GET | Current trained location metadata |
| `/api/location/set` | POST | Set new location, fetch NASA data, retrain |
| `/api/months` | GET | Month name list |
